# Data Layer (数据层)

## 目录职能划分
- **Shared/**: [真理契约] 存放前后端共用的 DTO、枚举、错误码、常量。这里的代码理论上可以 100% 镜像到后端项目。
- **Local/**: [本地私有] 存放仅客户端使用的本地持久化数据结构（如设置、偏好、本地缓存）。
- **Configs/**: [静态表] 存放策划配置表的 C# 定义类（通常由 Excel 导出）。

## 核心原则
1. **数据不包含逻辑**: Data 目录下的类应该是纯粹的 POCO (Plain Old CLR Objects)，仅包含字段和属性。
2. **区分 DTO 与 Model**: 这里的 DTO 负责传输，Models 目录下的类负责响应式展示。
